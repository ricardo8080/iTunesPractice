# itunestest

## Tutorial
```
Input the information necessary to search albums from your favorite authors, then click the search button, !checkout all of them!
```

## Employed Tools
```
Vue.JS, Jquery, Mocha + Chai, Html, Css 
```
## Project setup
```
npm install

npm run serve

npm run build

npm run test:unit
```
